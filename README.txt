Use the following Steps to setup our Recommendation System and
try out our Recommendation System.
-----------------------------------------------------------------------

1) Open CA_content_based_model.r script, change the working directory at Line 5,
and run through the entire code to generate 2 files.
(sim_mat_cos_desc.rds and sim_mat_cos_name.rds)

2) Open XXXX.r script, and run through the entire code to generate 1 files (XXXX.rds)

3) Lastly, open CA_Recommender_System.r script, change the working directory at Line 5
and run the code till Line 69 to complete the setup.

4) From Line 71 onwards, you can replace the value of the variables to demo different scenarios
using pre-created functions.
